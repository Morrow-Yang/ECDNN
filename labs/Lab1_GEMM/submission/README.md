# Lab 1 Submission Bundle

This directory is a clean submission bundle. Its contents map directly to the Lab 1 deliverables:

| File | Role |
| --- | --- |
| `Lab1_Report.pdf` | Submit this report after confirming the name and student ID. |
| `Lab1_Report.tex` | Editable LaTeX source for the report. Change `\studentname` and `\studentid` at the top, then compile with XeLaTeX. |
| `Q2_im2col_convolution.cpp` | Required Q2 C++ implementation. |
| `Q3_cache_optimized_gemm.cpp` | Required Q3 C++ implementation. |
| `Q1_results.csv` | Complete 108-row evidence for Q1: 27 independent I/J/K shapes times 4 algorithms. |
| `Q3_results.csv` | 32-run timing evidence used in the Q3 table. |

The required upload set is the PDF plus the Q2 and Q3 C++ files. Include `Q1_results.csv` and `Q3_results.csv` too when the submission system permits extra attachments.
